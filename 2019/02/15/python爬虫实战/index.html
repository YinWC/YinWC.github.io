思路

  • 目标：百度百科Python词条相关词条网页-标题和简介

  • 入口页：http://baike.baidu.com/view/21087.htm

  • URL格式：

    • 词条页面URL：/view/125370.htm

  • 数据格式：

    • 标题：

      • <dd class="lemmaWgt-lemmaTitle-title"><h1>***</h1></add>

    • 简介：

      • <div class="lemma-summary">***<div>

  • 页面编码：UTF-8
